# Getory
https://getory.netlify.app/

## 製品概要
言語ごとのリポジトリやランダムなリポジトリを簡単に発見できるサービス

## 特徴
- 多言語対応

## 使用技術
- Vue3(Nuxt3)
- TypeScript
- TailwindCSS
- vue-i18n
- Docker(docker-compose)
- GitHubAPI

## 対応ブラウザ
- Google Chrome
- Safari
- Firefox
- Microsoft Edge
